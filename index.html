About Me

언제나 세상에 도움이 되는 무언가를 만드는 개발자 조병규입니다.

저는 Node.js & TypeScript, AWS를 사용하는 주니어 개발자로 언제나 세상에 도움이 되는 새로운 것을 만들고자 합니다.

저는 인턴으로 재직하던 회사에서 쿼리 실행 계획을 통해 쿼리 최적화 하여 쿼리 실행 시간을 20초에서 4초 이하로 줄인 경험과 100만건 이상의 데이터를 호출 할 때 캐시와 스케줄러를 사용하여 api 호출 시간을 780ms에서 4ms 이하로 줄인 경험이 있으며 기존 수동으로 누락 데이터를 찾는 모니터링 시스템을 웹 스크래핑된 데이터를 바탕으로 누락이나 결함 데이터를 찾아내도록 자동화한 경험이 있습니다.

이전 회사에서는 천억 이상 매출 규모를 가진 모회사의 ERP 개발을 하였으며, 파일 업로드 및 다운로드를 AWS S3 스토리지 기반으로 개선하였습니다.

현재는 44억 규모의 투자를 받은 시리즈 A 리걸테크 스타트업에서 근무하고 있으며 법률문서 자동작성 서비스를 B2B 서비스로 커스터마이징 하여 기업에 서비스를 제공했고 LLM을 통한 기업문서 자동 체크리스트 기능 개발 하였습니다. 현재는 회사 메인 B2C 서비스를 개발하고 있습니다.

사이드 프로젝트 및 외주도 꾸준히 진행하면서 학습을 하고 있습니다. 유지보수가 용이한 코드를 위해 클린 코드를 작성하려 노력하고 로그를 자동으로 파일로 저장하여 관리하며 TDD로 프로젝트를 진행하고 있습니다. 배포 과정에 드는 수고를 줄이고 비즈니스 로직에 집중하기 위해 Docker와 github actions을 통해 배포를 자동화한 경험이 있습니다.
